Game Rules - Stay Cool Buddy!

Ziel des Spiels
  • Das Ziel des Spiels ist es, alle vier eigenen Spielfiguren als Erster vom Startfeld über das Spielfeld ins Ziel zu bringen.
Vorbereitung
  • Jeder Spieler wählt eine Farbe die Spielfiguren werden bei Spielstart platziert.
  • Ein zufälliger Spieler beginnt, danach geht es mit dem nächsten Spieler weiter.
Spielablauf
  • Jeder Spieler ist nacheinander an der Reihe und Würfelt
Starten
  • Ein Spieler muss eine 6 würfeln, um eine seiner Figuren aus dem Startfeld auf das Startfeld in seiner Farbe (das erste Feld des Rundkurses) zu setzen.
  • Wenn eine 6 gewürfelt wird, muss eine Figur aus dem Zuhause auf das Spielfeld gestellt werden.
  • Wenn alle Figuren im Zuhause sind, darf 3 mal gewürfelt werden.
  • Würfelt der Spieler eine 6, darf er nochmals würfeln.
  • Hat ein Spieler keine Figuren mehr im Startfeld, darf er auch mit den Figuren auf dem Spielfeld weiterziehen.
Ziehen
  • Nach dem Start zieht der Spieler seine Figur entsprechend der gewürfelten Augenzahl im Uhrzeigersinn vorwärts.
  • Landet eine Figur auf einem Feld, das von einer gegnerischen Figur besetzt ist, wird die gegnerische Figur zurück in das Zuhause gestellt ("schlagen").
  • Eigene Figuren können nicht geschlagen werden und dürfen auch nicht auf Felder mit eigenen Figuren gesetzt werden.
Ins Ziel ziehen
  • Wenn eine Figur das Spielfeld umrundet hat, geht sie in das Ziel. Dort kann sie nicht mehr geschlagen werden.
  • Beispiel: Steht eine Figur drei Felder vor dem Zielbereich und der Spieler würfelt eine 4, darf diese Figur ins Ziel ziehen.
  • Wenn alle Spielfiguren entweder Zuhause oder im Ziel sind, darf wieder 3 mal gewürfelt werden.
Spielende
  • Das Spiel endet für jeden Spieler, wenn alle vier Figuren in eigener Farbe den Zielbereich gebracht wurden. Dieser Spieler ist der Gewinner.
  • Die restlichen Spieler spielen darum, nicht letzer zu werden. Wer als letztes noch auf dem Feld steht, hat verloren
